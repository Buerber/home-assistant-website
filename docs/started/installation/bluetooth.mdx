# Bluetooth

<p className="text-xl">The **Bluetooth** integration will detect nearby Bluetooth devices. Discovered devices will show up in the discovered section on the integrations page in the configuration panel.</p>

## Configuration 

To add the **Bluetooth** integration to your Home Assistant instance, use this My button:

[添加 integration](https://my.home-assistant.io/redirect/config_flow_start?domain=bluetooth)
