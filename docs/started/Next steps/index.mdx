# 下一步操作

<p className="text-xl font-semibold">入门过程带您完成了 Home Assistant 的初始设置，例如启动系统、命名您的家庭和选择位置。本节将引导您了解进一步的文档，帮助您进行下一步操作。</p>

## 向 Home Assistant 添加其他人员

您可以向 Home Assistant 添加其他人员。他们可以拥有自己的登录账号，在他们的设备上使用 Home Assistant 并创建自己的仪表板。要添加其他人员，请参考向 Home Assistant 添加人员。

## Android 和 iOS 应用

您可以在手机、智能手表甚至汽车上使用 Home Assistant。

- 要了解如何在 Android 或 iOS 上安装 Home Assistant，请参考[配套应用的文档](https://companion.home-assistant.io/)。

- 想使用语音控制 Home Assistant？
    - 请参考[在 Android 上使用 Assist](https://www.home-assistant.io/voice_control/android/)的文档。

## 更改基本设置

要更改位置、单位系统和语言等基本设置，请参考[更改基本设置](https://www.home-assistant.io/docs/configuration/basic/)。

## 创建备份

您可以备份您的 Home Assistant、插件数据和配置。备份用于在需要回滚时恢复系统或其部分，或将您的 Home Assistant 迁移到新硬件。在更新前创建备份是一个好习惯。

要了解如何创建 Home Assistant 安装的备份，请选择与您的安装类型匹配的链接。

- Home Assistant 操作系统：[从操作系统创建备份](https://www.home-assistant.io/common-tasks/general/#backups)
- Home Assistant Supervised：[从 Supervised 创建备份](https://www.home-assistant.io/common-tasks/general/#backups)
- Home Assistant Core 或 Home Assistant 容器：[从 Core 或容器创建备份](https://www.home-assistant.io/integrations/backup/)

## 编辑 configuration.yaml 和配置文件访问

虽然您可以从用户界面配置 Home Assistant 的大部分功能，但对于某些集成，您需要[编辑 `configuration.yaml` 文件](https://www.home-assistant.io/docs/configuration/)。此文件包含要加载的集成及其配置。在整个文档中，您将找到可以添加到配置文件中以启用特定功能的代码片段。对于初学者来说，不需要编辑 `configuration.yaml` 文件。当需要时，您会被引导到相关文档。

## 设置网络存储

如果您需要更多空间来存储数据，可以配置[网络存储](https://www.home-assistant.io/common-tasks/os/#network-storage)，例如用于存储备份或访问媒体。

## 开始使用语音助手

如果您想开始使用语音助手，请参考[Assist](https://www.home-assistant.io/voice_control/)的文档。
