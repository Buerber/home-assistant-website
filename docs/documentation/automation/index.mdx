# 自动化 Home Assistant

Home Assistant 包含了您所有设备和服务的信息。这些信息可以在仪表板中查看，也可以用于触发自动化。这非常有趣！

Home Assistant 中的自动化允许您自动响应发生的事情。您可以在日落时打开灯光，或者在接到电话时暂停音乐。

如果您刚刚开始，我们建议您从蓝图自动化开始。这些是由社区提供的现成自动化，您只需要进行配置。
